# Mehmet-Bank-ATM-Simulation
Python ile geliştirilmiş, Nesne Yönelimli Programlama (OOP) ilkelerine dayalı, dinamik limit ve güvenlik duvarı barındıran gelişmiş bir ATM Simülasyonu.
# 🏦 Mehmet Bank - Gelişmiş ATM Simülasyonu

Bu proje, Python dilinin temel mantığını ve **Nesne Yönelimli Programlama (OOP)** ilkelerini pekiştirmek amacıyla geliştirilmiş bir konsol (terminal) uygulamasıdır.

## 🚀 Öne Çıkan Özellikler
* **Dinamik Kredili Mevduat Hesabı (Eksi Bakiye):** Kullanıcının ana parasına göre eksi bakiye limiti otomatik belirlenir ve eksiye düşüldüğünde bu tutar borç olarak kaydedilir.
* **Siber Güvenlik Simülasyonu:** Şifremi unuttum ekranında `random` kütüphanesiyle 6 haneli geçici SMS kodları üretilir ve kimlik doğrulaması yapılır.
* **Gelişmiş UI/UX:** ANSI renk kodlarıyla renklendirilmiş terminal tasarımı ve dinamik ekran temizleme özellikleri mevcuttur.
